# Deployment

CodeSnap JS V4 is a static project and does not need a backend or build step.

## GitHub Pages

1. Upload all files to the repository root.
2. Open repository Settings.
3. Go to Pages.
4. Select `Deploy from a branch`.
5. Choose `main` and `/ root`.
6. Save.

The site will be available at:

```text
https://username.github.io/repository-name/
```

## Important

Do not move `index.html` into a subfolder. It must stay in the project root.
