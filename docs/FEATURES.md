# Features

## Snippet Studio

Create, edit, favorite, archive, copy, export, and version snippets.

## Code Playground

Run small HTML, CSS, and JavaScript experiments inside a sandboxed browser preview.

## Notes

Write developer notes and link them with saved snippets.

## Collections

Group snippets into larger packs such as React Hooks, CSS Components, API Examples, or Learning Notes.

## Tag Manager

Rename, remove, and filter tags across snippets.

## Backup

Export full workspace data as JSON and import it later.
